# cse210-ww-student-template
This is the starter code for students in the worldwide content version of CSE 210.

This repository contains the starter code for many different projects. They are arranged folders by week. There is also an an empty sandbox project that can be used to play around with any concept you would like.

## Projects

### Week 02 - Resumes
A program that creates and displays a resume with job experiences.
- **Concepts:** Classes, Objects, Abstraction
- **Files:** `Job.cs`, `Resume.cs`, `Program.cs`

### Week 03 - Scripture Memorizer
A program to help memorize scriptures by progressively hiding words.
- **Concepts:** Encapsulation, Private Member Variables, Constructors
- **Features:**
  - Random word hiding
  - Scripture library with 5+ scriptures
  - Loop to memorize multiple scriptures without restarting
- **Files:** `Scripture.cs`, `Reference.cs`, `Word.cs`, `Program.cs`

### Week 03 - Fractions
A program to work with fractions and display them as fractions or decimals.
- **Concepts:** Encapsulation, Multiple Constructors, Getters
- **Files:** `Fractions.cs`, `Program.cs`
